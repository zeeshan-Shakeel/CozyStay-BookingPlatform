# 🏡 CozyStay – Full Stack Rental Listing Web Application
## 📌 Overview

**CozyStay** is a modern rental listing platform that allows users to browse, post, and review properties for rent. Built with a powerful Node.js backend, EJS templating, and MongoDB Atlas for scalable data storage.

---

## 🚀 Features

- 🏠 Create, Read, Update, Delete (CRUD) listings
- 🔐 User authentication & sessions via **Passport.js**
- 🌐 Cloud-based image uploads via **Cloudinary**
- 📍 Map integration with **Mapbox**
- 📄 Flash messages and form validation
- 🎨 Responsive UI using **Bootstrap 5**
- 📬 Secure contact form and privacy policy

---

## 🛠 Tech Stack

### 🔧 Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Passport.js (LocalStrategy)

### 🎨 Frontend
- EJS (Embedded JavaScript)
- Bootstrap 5
- Font Awesome

### 📦 Cloud & APIs
- Cloudinary for image hosting
- MApTiler for location maps
- .env for environment config
